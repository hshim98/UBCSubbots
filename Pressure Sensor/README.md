How to connect the pressure sensor:

On the arduino: 

-Red Wire goes to 5V or 3.3V
-Black wire goes to ground
-White wire connected to SDA (important)
-Green wire connected to SCL (important)

If you get an error make sure that the SDA and SCL wires are properly connected.

Library needed to download:
-BlueRobotics MS5837 
  This can be downloaded via Library Manager (Sketch–>Include Library–>Manage Libraries)

There is an example code that can be ran to ensure the connections are right. Assuring connections are right the pressure sensor is ready to be used.
